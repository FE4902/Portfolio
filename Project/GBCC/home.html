Compensation Formula

The TITANIUM:MILICO ratio is 10,000 to 1.
The number of days the server has been opened for, MILICO’s market value, the formula, and fee are all calculated.
In MILICO creation, the decimal points of the number of TITANIUM needed for 1 MILICO are rounded down to zero.

*The formula is updated every day at 00:00 (UTC+0).

  • Server Day: Day of Server Creation (Days)
  • M(t-1): The Average MILICO Market Value of the Previous Hour (E.g: If it’s currently 12:24, the average MILICO market value of 11:00 - 11:59)
  • M(d-1avg): The Average MILICO Market Value of the Previous Day (E.g: If it’s currently 2/8 11:50, the average MILICO market value of 2/7)
  • SAVα, SAVβ: A value calculated with: the period of how long a server has been open for, number of active users, and amount of cumulated MILICO creation count
